# RegreLin

RegreLin is a function I wrote in Python for linear regression using least squares with two variables.

Explanation:

RegreLin: given points defined by cartesian coordinates xx and yy, exist a straight line defined as y=mx+by=mx+b, that reduce at minimum the Euclidean distance to all of them.

The function returns the slope mm and the y-axis intercept bb of the straight line. The determination coefficient r−squarer−square is also returned.

If those interested in understanding how works linear regression with least squares, and examples, consult the books (in spanish) "Fundamentos de probabilidad y estadística" (first ed.) by Jay L. Devore (2018) or "Estadística" (12 ed.) by Mario F. Triola (2018)
